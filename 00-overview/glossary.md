
# Glossary

**Menu Service**  
Microservice responsible for managing menu items, sections, categories, availability, pricing, and images for each cafe.

**Order Service**  
Microservice responsible for creating, updating, tracking, and processing customer orders for cafes, including real-time status updates.

**Customer**  
User placing an order through the system.

**Cafe**  
The venue or restaurant using SmartCafe to manage menus and orders.

**Menu Item**  
Product that customers can order. Includes name, description, price, category, images, and ingredient options.

**Menu Category**  
Label assigned to a menu item (e.g., Vegetarian, Spicy). Items can have multiple categories; categories may have icons/images and can be configured by the cafe.

**Menu Image**  
Visual representation of a menu item. Includes big image (main display), cropped image (for fast scrolling), and default image if none provided.

**Order**  
Request made by the customer for one or more menu items.

**Order Status**  
Represents the current state of an order (e.g., Pending, Accepted, Preparing, Ready, Delivered, Cancelled).
